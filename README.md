# Cine-DB

Un sistema moderno de gestión de cines desarrollado en **Python** con interfaz gráfica **Tkinter** y almacenamiento no relacional en **MongoDB**. 

El sistema permite administrar de forma eficiente el catálogo de películas, la programación de funciones en salas, el control de usuarios y la venta automatizada de entradas con actualización de inventario en tiempo real.

## Instalación

Para instalar las dependencias necesarias, ejecuta el siguiente comando:

```bash
pip install pymongo
```

## Cómo Iniciar

Para iniciar el sistema, ejecuta el siguiente comando:

```bash
python main.py
o
py main.py
```

## Tecnologías Utilizadas
* **Lenguaje:** Python 3.x
* **Base de Datos:** MongoDB (NoSQL)
* **GUI:** Tkinter
* **Driver:** PyMongo
